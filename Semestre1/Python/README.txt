    
    Seja bem-vindo ao Kobayashi Combat,

Um jogo inspirado na mecânica do jogo Asteroids de 1979 (https://en.wikipedia.org/wiki/Asteroids_(video_game))
onde o jogador não só tem movimento no eixo horizontal (como inicialmente planejado similar ao Space Invaders), 
mas no eixo vertical também. A diferença se encontra que no topdown 2D implementado, as naves te seguem, e no Asteroids não.
Diferentemente do Asteroids, ainda, que só permitia impulso para frente, nesse minigame você pode mover para todos os lados. 
O que é mais necessário quando tem 10 naves Klingon atrás de você. 

Todos os assets foram feitos usando Aseprite. A música do jogo é um trecho do seguinte vídeo: https://www.youtube.com/watch?v=VrXeBXOU_1Q&t=107s
Esse jogo é inspirado no professor falando sobre Star Trek, que é minha série favorita. 

Para iniciar o jogo, favor fazer no terminal: Python3 inicio.py
